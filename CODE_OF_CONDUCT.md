# Contributor Covenant Code of Conduct

## Pledge

Community committed to providing welcoming, inclusive experience for all. Respect, kindness, professionalism expected.

## Standards

Positive behavior:
- Respectful communication
- Accept constructive criticism
- Focus on community benefit
- Show empathy

Unacceptable behavior:
- Harassment, insults, derogatory comments
- Discrimination based on protected characteristics
- Sexual harassment or unwelcome advances
- Doxxing or privacy violations
- Spam, off-topic disruption
- Trolling

## Enforcement

Maintainers enforce Code of Conduct. Violations reported via:
- GitHub issues (non-urgent)
- Email (sensitive concerns)

Enforcement scale: warning → temporary mute → permanent ban.

## Scope

Applies to all project spaces: GitHub, discussions, code reviews, public communications.
